## WHAT'S INCLUDED

### 📱 **6 Web Pages**
1. **Homepage** - Information hub with links to all sections
2. **Classes & Events** - Current and upcoming class schedules with registration
3. **Class Materials** - Repository of prayers and slokas (Tamil, Sanskrit, English)
4. **Coordinator Login** - Secure entry point
5. **Allocation Tool Portal** - Access to three fair allocation tools
6. **Photo Gallery** - Placeholder ready for photos

### 🎯 **3 Advanced Allocation Tools**
- **VSN Tool** (Viṣṇu Sahasranāma) - Distributes chanting segments fairly
- **Lakshmi Tool** (Lakṣmī Sahasranāma) - Same functionality
- **Lalitha Tool** (Lalitā Sahasranāma) - Same functionality

**Each tool includes:**
- ✅ Round-robin segment assignment
- ✅ Random opening/closing prayer selection
- ✅ Excel export (2 formats)
- ✅ Professional PDF export
- ✅ Works on desktop, tablet, and mobile

### 🔒 **Security Features**
- Password-protected coordinator access
- Secure session management
- No external data sharing
- HTTPS encryption (GitHub Pages)
